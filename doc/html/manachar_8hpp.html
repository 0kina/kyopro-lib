Detailed Description

Manacharアルゴリズムに関するコード。

  • 文字列のi文字目を中心とした回文の最大半径(i文字目を含む)を合計O(N)で求める。
  • 同じ時間でi文字目とi+1文字目の間を中心とする回文の最大半径を求めることもできる。
